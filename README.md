
# Let's Build

Source code for:

https://letsbuild.cloud

## Installation and pre-requisites

Install tailwind globally:

```sh
npm i -g tailwindcss
```

## How to build

Generate new CSS:

```sh
npx tailwindcss -i css/input.css -o css/output.css
```

## How to deploy

Commit and push, it will deploy automatically.
